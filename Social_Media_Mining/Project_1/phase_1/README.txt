
Execution instructions:

Run with command, python phase_1.py 

The following files will be generated inside phase_1_output folder.

anonymized_edge_list.txt - Contains anonymized list of edges separated by commas. 
anonymized.txt - Contains the mapping between actual member ids and anonymized ids.
edge_list.txt - Contains the edge list obtained through crawling.
anonymized_sampled.txt - Contains anonymized sampled nodes.
sampled.txt - Contains sampled nodes.
